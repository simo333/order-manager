# Orders Manager
Web application that manages production company. It manages products, employees, orders etc. Based on true production company.

### FEATURES:
* Manage employees
* Manage models and their types

### USED TECHNOLOGIES:
* Java 17
* Spring 2.5.6
* Hibernate 5.1.2
* MySQL
* Angular 13 (TypeScript 4.5.5)
* HTML / CSS (Bootstrap)
